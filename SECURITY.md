# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.0.x | ✅ 支持 |
| < 1.0  | ❌ 不支持 |

## 报告安全漏洞

如果你发现安全漏洞，请直接发送邮件至：
- security@tensorslab.com

请提供：
- 漏洞描述
- 受影响的版本
- 复现步骤
- 可能的解决方案

我们会在 48 小时内回复，并尽快修复。

## 安全最佳实践

### API Key 管理
- 永远不要将 API Key 提交到仓库
- 使用环境变量存储敏感信息
- 定期轮换 API Key

### 用户数据
- 本地处理用户输入
- 不要收集和上传用户数据
- 遵循隐私保护原则

## 依赖安全

我们定期检查和更新依赖，确保：
- 使用最新稳定版本
- 修复已知漏洞
- 最小化依赖范围

## 安全更新

我们会及时发布安全更新，请：
- 关注 Release 说明
- 及时更新到最新版本
- 查看 CHANGELOG 了解变更
